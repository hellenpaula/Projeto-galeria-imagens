## 📌 Sobre:

Este é um projeto de galeria visual desenvolvido com HTML e CSS, com foco na prática e criação de layouts modernos e responsivos utilizando CSS Grid.

A proposta do projeto é apresentar diferentes estilos arquitetônicos de residências ao redor do mundo, destacando características como design, iluminação e integração com o ambiente.

Durante o desenvolvimento, foram trabalhados conceitos importantes como organização de layout em grid, posicionamento de elementos, responsividade e estruturação semântica do HTML.


## 🧩 Funcionalidades:

- Exibição de uma galeria de casas com diferentes estilos arquitetônicos.

- Organização dos elementos em um layout com CSS Grid.

- Alternância entre imagens e descrições para criar uma experiência visual fluida.

- Layout responsivo adaptado para diferentes tamanhos de tela:

  - Mobile

  - Tablet

  - Desktop

## 💻 Tecnologias utilizadas:

- [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) - Estrutura semântica do conteúdo.

- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) - Estilização, layout com Grid.

- [Media Queries](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_media_queries/Using_media_queries) - Responsividade da página.

- [Google Fonts](https://fonts.google.com/) - Tipografia utilizada no projeto.


## ⚙️ Estrutura e organização do layout:

O projeto é estruturado em duas partes principais:

### 1️⃣ Área de introdução

- Contém o título principal e a descrição do projeto.

- Posicionada de forma estratégica para apresentar o contexto antes da galeria.


### 2️⃣ Grid de conteúdo

- Utiliza CSS Grid com grid-template-areas para organizar os elementos.

- Combina:

  - Cards de imagem

  - Blocos de texto

  - O layout alterna entre imagem e descrição para criar ritmo visual.

### 📐 Responsividade

 Foram utilizadas **media queries** para adaptar o layout:

- Mobile (padrão):

  - Grid com 2 colunas

  - Layout mais compacto e vertical

- Tablet (min-width: 830px):

  - Aumento do número de colunas

  - Reorganização dos elementos no grid

- Desktop (min-width: 1300px):

  - Layout mais sofisticado

  - Melhor aproveitamento de espaço

  - Ajustes finos de posicionamento

## 🧠 Aprendizados:

Durante o desenvolvimento deste projeto, pude reforçar e praticar:

- Uso de CSS Grid (grid-template-areas, rows e columns).

- Posicionamento de elementos com position: absolute e relative.

- Criação de layouts responsivos com media queries.

- Construção de interfaces com foco em design visual.
